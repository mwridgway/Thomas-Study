# ⚡ The Intersect — Thomas's Study Hub

Grade 12 NSC revision web app — built for Agent Bartowski @ Paul Roos Gimnasium.

> *"Guys, I know kung fu."* — Except it's Biology.
>
> Maintained by Sarah Walker 🕶️ | Chuck-themed because we're Nerd Herd.

## Subjects

- 🌍 **Geography** — Mid-Latitude Cyclones, Tropical Cyclones (40 flashcards, 30 quiz questions, 10 Feynman topics)
- 🧬 **Life Sciences** — DNA, RNA & Meiosis (58 flashcards, 45 quiz questions, 14 Feynman topics)
- 💻 **CAT** — System Technologies (flashcards, quiz, Feynman)

## Features

- 📖 **Structured Notes** — Full content broken into digestible sections
- 🃏 **Active Recall Flashcards** — Flip cards with confidence tracking
- ✏️ **NSC-Style Quizzes** — Multiple choice with detailed explanations
- 🧠 **Feynman Mode** — Explain concepts in your own words, then check key points
- 📅 **Term Schedule** — Filter by subject, term, and assessment type

## Tech

Static HTML/CSS/JS. Deployed via GitHub Pages.
